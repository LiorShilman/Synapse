# הצעה לשילוב מערכת סינאפס — פלטפורמת אינטליגנציה קולקטיבית מבוססת AI פנימי

## תקציר מנהלים

מערכת **סינאפס** היא פלטפורמה ויזואלית לפתרון בעיות מורכבות באמצעות 6 סוכני AI מתמחים הפועלים בשיתוף פעולה. המערכת מנתחת דילמות ממספר זוויות בו-זמנית — ניתוח נתונים, אימות לוגי, חשיבה יצירתית, זיכרון ארגוני, תיאום עמדות וסינתזה — ומגיעה לקונצנזוס מנומק.

**ההצעה:** לשלב את המערכת עם ה-LLM הפנימי הקיים בארגון, וליצור כלי חשיבה קולקטיבי זמין לכלל העובדים — ללא תלות באינטרנט חיצוני, תוך שמירה מלאה על אבטחת מידע.

## ארכיטקטורת השילוב

```
┌─────────────────────────────────────────────┐
│            רשת פנימית של הארגון              │
│                                             │
│  ┌──────────┐    API    ┌───────────────┐   │
│  │  סינאפס  │ ◄──────► │  LLM פנימי    │   │
│  │  (React) │          │  (On-Premise)  │   │
│  └──────────┘          └───────────────┘   │
│       │                                     │
│       ▼                                     │
│  ┌──────────┐                               │
│  │  דפדפן   │  ← כל עובד ניגש דרך הדפדפן  │
│  │  העובד   │                               │
│  └──────────┘                               │
└─────────────────────────────────────────────┘
       ❌ אין חיבור לאינטרנט חיצוני
```

### מה נדרש טכנית

1. **שרת פנימי** — להרצת אפליקציית סינאפס (Node.js/Nginx, משאבים מינימליים)
2. **חיבור API ל-LLM הפנימי** — התאמה של נקודת הקצה (`endpoint`) במקום Claude API
3. **ללא תלות חיצונית** — הכל רץ on-premise, אין קריאות החוצה

### שינויים נדרשים במערכת

- החלפת `VITE_ANTHROPIC_API_KEY` בחיבור ל-API הפנימי
- התאמת פורמט הבקשה/תגובה לפי ה-LLM שבשימוש
- (אופציונלי) הוספת מערכת הזדהות פנימית (SSO/LDAP)

## 6 הסוכנים — איך כל אחד תורם

| סוכן | תפקיד | ערך לארגון |
|-------|--------|-----------|
| 🔮 **אורקל** — אנליסט ראשי | ניתוח נתונים, זיהוי מגמות, הערכות כמותיות | החלטות מבוססות data, לא תחושות בטן |
| 🔗 **נקסוס** — מרכז תקשורת | חיבור בין תובנות, תיאום עמדות | מונע silos — מחבר נקודות מבחינות שונות |
| ⚡ **פורג'** — ממזג רעיונות | פתרונות יצירתיים, חשיבה מחוץ לקופסה | מעודד חדשנות, מציע מה שאף אחד לא חשב |
| 📡 **אקו** — זיכרון קולקטיבי | זיהוי דפוסים, למידה מהעבר | מונע חזרה על טעויות, בונה ידע מצטבר |
| 🔐 **סייפר** — מאמת לוגי | ספקן קונסטרוקטיבי, בדיקת הנחות | מזהה כשלים לפני שהם קורים |
| 🧠 **סייג'** — מסנתז חוכמה | מסקנות סופיות, אינטגרציה | ממקד את כל הדיון לתוצאה ברורה ופעילה |

- תכנון מעבר בין טכנולוגיות

- הערכת ספקים וכלים
- ניהול שינויים ארגוניים
- חלוקת משאבים בין צוותים

### 👤 אישי-מקצועי (לכל עובד)
- התלבטות בין מסלולי קידום
- תכנון למידה והתפתחות מקצועית
- איזון עבודה-חיים
- החלטות כלכליות אישיות

## יתרונות עיקריים

### 🔒 אבטחת מידע
- **אפס חשיפה חיצונית** — הכל רץ ברשת הפנימית
- **ללא העברת מידע לענן** — ה-LLM פנימי, הנתונים פנימיים
- **תואם מדיניות אבטחה קיימת** — אין צורך בפתיחת פורטים או חריגים

### ⚡ ערך מיידי
- **פריסה מהירה** — אפליקציית web סטטית + חיבור API, ימים ולא חודשים
- **אפס הדרכה** — ממשק אינטואיטיבי בעברית, מדריך מובנה במערכת
- **10 קטגוריות מוכנות** עם 40 דוגמאות — העובד מתחיל בלחיצה

### 📊 שקיפות מלאה
- כל תהליך החשיבה **גלוי בזמן אמת** — לא "קופסה שחורה"
- **ויזואליזציה תלת-ממדית** של הרשת הנוירונית
- **דוחות HTML מרשימים** להורדה ושיתוף עם קולגות

### 🎯 קבלת החלטות טובה יותר
- **6 נקודות מבט** במקום אחת
- **אימות לוגי אוטומטי** — לא רק "מה לעשות" אלא "למה"
- **זיהוי נקודות עיוורות** שאדם בודד עלול לפספס

## סיכום

מערכת סינאפס מציעה דרך חדשה לקבל החלטות בארגון — לא עוד "שאל את ChatGPT", אלא **צוות מומחים וירטואלי** שמנתח כל בעיה מ-6 זוויות, מתווכח, מגיע לקונצנזוס, ומייצר תוצאה מנומקת ומתועדת.

בשילוב עם ה-LLM הפנימי הקיים, המערכת הופכת **מכלי הדגמה לכלי ייצור** — זמין לכל עובד, מאובטח לחלוטין, וללא עלות שוטפת.

**הצעד הבא:** תיאום הדגמה חיה למנהלים → אישור POC → שבועיים לפריסה ראשונית.
